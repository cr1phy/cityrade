# CityRade

CityRade - это игра, написанная на языке программирования Rust.

## Описание

CityRade - это увлекательная игра, где игроки могут строить и управлять своим городом. Ваша задача - развивать инфраструктуру, управлять ресурсами и населением, чтобы создать процветающий город.

## Основные особенности

- **Строительство:** Постройте различные здания и сооружения, чтобы улучшить ваш город.
- **Управление ресурсами:** Управляйте ресурсами, такими как еда, вода и энергия, чтобы обеспечить устойчивое развитие.
- **Экономика:** Развивайте экономику города, создавая рабочие места и привлекая новых жителей.
- **События:** Реагируйте на различные события и вызовы, чтобы сохранить стабильность и процветание вашего города.

## Установка и запуск

### Требования

- [Rust](https://www.rust-lang.org/) 1.56 или выше

### Установка

1. Клонируйте репозиторий:
    ```bash
    git clone https://github.com/cr1phy/cityrade.git
    ```

2. Перейдите в директорию проекта:
    ```bash
    cd cityrade
    ```

3. Соберите проект:
    ```bash
    cargo build --release
    ```

### Запуск

Запустите игру с помощью следующей команды:
```bash
cargo run --release
```
